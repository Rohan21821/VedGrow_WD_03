# Smart Quiz Master

An interactive and responsive Quiz Application built using HTML, CSS, and JavaScript. The application fetches real-time questions from the Open Trivia Database (Open Trivia DB) API and provides an engaging quiz experience with timer-based questions, score tracking, performance analytics, and answer review functionality.

## Features

- Real-time quiz questions from Open Trivia DB API
- Category selection
- Difficulty selection (Easy, Medium, Hard)
- 10, 15, and 20 question modes
- 15-second timer for each question
- Dynamic progress bar
- Correct and incorrect answer tracking
- Accuracy percentage calculation
- Streak and Best Streak tracking
- Best Score saved using Local Storage
- Light and Dark Theme support
- Answer Review section after quiz completion
- Sound effects for correct and wrong answers
- Loading and Error Handling screens
- Fully responsive design for desktop and mobile devices
- Personalized user experience with player name support

## How to Run

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Enter your name.
4. Select category, difficulty, and question count.
5. Start the quiz and test your knowledge.
